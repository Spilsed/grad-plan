# Grad-Plan

It's difficult to plan more than a few semesters in the future without making mistakes. Having to handle all the categories of degree requirements and the course prerequisites to get you there. Grad-Plan allows you to enter a your course plan for your degree and will determine if it is a valid path to take. If something is wrong with your plan it is able to tell you every possible class that is able to make it valid, instead of just the constraints for the class.

## Quick Start

### Requirements

- Python 3.12 or above
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

### Setup

1. Clone the repository
2. uv sync
3. Source the created .venv
4. Run main.py where it will request your plan
